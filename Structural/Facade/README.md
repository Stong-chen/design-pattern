# 外观模式(门面模式)

Facade

## 解释

就是把一个复杂的问题简单的给到用户。

例如我买了一套房子，我要装修。装修的内容可就很多了。什么水管啊，电线啊，墙面啊，窗户啊。乱七八糟的。

用户其实不需要懂这么多东西。所以我们给用户提供一个简单的门面，也就是我们的装修公司。直接找他，说我要装修成什么样。直接给钱就完事。

用户不用去了解装修的细节。
